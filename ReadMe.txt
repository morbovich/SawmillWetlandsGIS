Shapefiles for Sawmill Wetland Education Area.

Wetland Polygon
Boardwalk + Nature Trail Line
Piezometer Location Point
Curb Cut Point
Berm Centerline
Pool A,B,C Polygons

**DEM data (ned19 files) taken from USGS. Manipulated dataset not included. go to USGS.gov to download DEM file and upload into ArcGIS as a .tiff

Download all supporting files with shapefile when opening in ArcGIS (i.e. .cpg, .dbf, .prj, etc. to ensure geolocation is accurate and drawings are accurate
